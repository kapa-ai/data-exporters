# Data Exporters

Learn more about getting started with exporting data from platforms that are currently unsupported by the existing data source system to a format that can be ingested into [Kapa.ai](https://kapa.ai/) for additional context in your conversations.

## Exporters

- [Pylon](./pylon-exporter/)
- [Linear](./linear-exporter/)

### Disclaimer 

APIs can change over time which may break these exporters, if you noitce any issues with these exporters, reach out to the support team! Scripts are often made to be general-purpose and may need to be modified by yourself, or your team, to fulfil organisation-specific requirements.
